# EASDavisDashboard-045
PROJECT AKHIR DATA VISUALISASI B 

AI-Augmented Sales Dashboard

Project ini merupakan tugas EAS Data Visualization (DAVIS B) yang bertujuan untuk membangun dashboard analisis data penjualan berbasis visualisasi interaktif dan insight otomatis.

Dashboard ini tidak hanya menyajikan data dalam bentuk grafik, tetapi juga memberikan insight dan rekomendasi bisnis untuk membantu pengambilan keputusan yang lebih cepat dan akurat.

**Objectives**
1. Menyajikan KPI utama performa penjualan
2. Menganalisis pola data penjualan dari berbagai kategori
3. Menyediakan insight berbasis data
4. Memberikan rekomendasi bisnis sederhana
5. Membuat dashboard interaktif dan mudah dipahami

**DATASET**
Data yang digunakan merupakan Dataset Sales by Category yang telah di lalui tahap cleaning. Data terdiri dari 
1. Sales
2. Profit
3. Category & Sub  Category
4. Order Date
5. Teritory


 **Tech Stack**
HTML, CSS, JavaScript
Tableau Public (visualization)
Netlify (deployment)

📌 Notes
Project ini dibuat sebagai bagian dari pembelajaran data visualization dan implementasi dashboard berbasis insight untuk mendukung analisis bisnis sederhana.
